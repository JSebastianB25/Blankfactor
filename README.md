Blankfactor QA Automation Framework

Technologies Used

- Cypress 15.8.2
- Cucumber (Gherkin) scenario definition.
- Cucumber HTML Reporter - For advanced visual reporting.
- JavaScript 22.16.0 - Programming language.

Installation
npm install
npm install cypress --save-dev
npm install --save-dev @badeball/cypress-cucumber-preprocessor @bahmutov/cypress-esbuild-preprocessor
npm install --save-dev cucumber-html-reporter

Execution
HTML Cucumber report
- npm test
